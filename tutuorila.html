PYTHON



Python is a high-level, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation. Python is dynamically typed and garbage-collected. It supports multiple programming paradigms, including structured, object-oriented and functional programming. Wikipedia Designed by: Guido van Rossum
First appeared: 20 February 1991; 31 years ago
OS: Windows, macOS, Linux/UNIX, Android and more
Filename extensions: .py,.pyi,.pyc,.pyd,.pyw,.pyz (since 3.5),.pyo (prior to 3.5)
Paradigm: Multi-paradigm: object-oriented, procedural (imperative), functional, structured, reflective
Preview release: 3.12.0a5 / 7 February 2023; 2 days ago
? Parent company: Python Software Foundation

C ADN C++

C++ is a cross-platform language that can be used to create high-performance applications.
C++ was developed by Bjarne Stroustrup, as an extension to the C language.
C++ gives programmers a high level of control over system resources and memory.
The language was updated 4 major times in 2011, 2014, 2017, and 2020 to C++11, C++14, C++17, C++20.
